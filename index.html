Tizen Video App

Video player based on Tizen webapis.avplay (without drm support)

Available buttons/actions:

1 - Toggle UHD content playback - use before starting playback and only for 4K content
2 - display track info - while playback active
3 - display stream properties - while playback active

Info

This application demonstrates the use of AVPlay plugin as video player

The application downloads video content from http://playready.directtaps.net - it will not work if this domain is not accessible
You can change the source for video content by modifying url of config variable in main.js file

Operation

Use mouse or remote to operate on video player.
Available keys: PLAY, PAUSE, FF, RW, STOP, ENTER (Toggle Fullscreen mode), RETURN
